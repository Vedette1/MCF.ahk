# MCF (MCode Forge)

MCF (MCode Forge) — это автономный COFF-линковщик, реализованный на AutoHotkey v2 и предназначенный для самой платформы AutoHotkey v2.

![Screenshot](https://github.com/Vedette1/MCF.ahk/blob/main/screenshot1.png)

## О проекте

Основная цель проекта — это простота сборки Mcode из исходников на C. Пишите обыденный C-код и в один клик кнопки получите готовый Mcode.

## Ключевые возможности и встроенные утилиты

- Встроенный COFF-линковщик.
Полный цикл обработки объектных файлов: разбор COFF-заголовков, обработка таблиц символов и релокаций, вычисление относительных адресов и генерация финального позиционно-независимого машинного кода.

- Унифицированный GUI.
Графическая оболочка выполняет роль фронтенда для всего тулчейна и позволяет запускать компиляцию и линковку без переключения в командную строку.

- Интеграция с компиляторами.
Поддерживаются Microsoft Visual C++ (MSVC) и GCC (MinGW). Выберите компилятор и флаги через GUI, после чего проект в один клик кнопки соберет объектный файл, необходимый для дальнейшей линковки.

- Дизассемблер и просмотр бинарных данных
В GUI встроена поддержка objdump для быстрого анализа исходного кода, а также hex-вьюер. Пока что эти функции находятся в бете и в дальнейшем будут улучшаться.

## Текущие ограничения

Линковщик не поддерживает C Runtime Library (CRT). Код, использующий стандартные функции C, такие как printf, в данный момент не может быть обработан корректно. Причина заключается в том, что CRT требует компоновки со статическими библиотеками, парсер для которых пока не реализован.
